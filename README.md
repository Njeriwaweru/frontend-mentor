# Frontend Mentor Challenges

This repository contains a collection of frontend projects built from challenges on **Frontend Mentor**.  
Each project focuses on improving HTML, CSS, and JavaScript skills through real-world UI layouts and components.

The goal of this repository is practice, experimentation, and steady improvement in frontend development.
